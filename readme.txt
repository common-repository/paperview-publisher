This plugin integrates your WordPress instance with your Paperview Publisher account, using the Paperview public API.

== Description ==

Learn more about Paperview [here](https://paperview.net).
Create your Paperview Publisher account at the [paperview.cloud](https://paperview.cloud) website.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/paperview-publisher` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Paperview->Publisher settings screen to configure the plugin
1. If you wish to use only one Paperview Publisher editor user, then you're good to go! Otherwise, you need to go into the corresponding WordPress user's profiles, and enter their Paperview User ID in the corresponding field.

== Frequently Asked Questions ==

= Who is this for? =

This is a plugin meant for Paperview Publishers (namely, content editors). It allows editors to use Paperview in their publications, in a manner that is easier and more expedite than plain copy-pasting article content back and forth between the Paperview website and their own WordPress publication.
